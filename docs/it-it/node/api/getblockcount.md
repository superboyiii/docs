# Metodo getblockcount

Ottiene il numero di blocchi nella chain principale.

## Esempio

Corpo della richiesta:

```json
{
   "jsonrpc": "2.0",
   "method": "getblockcount",
   "params":[],
   "id": 1
}
```

Corpo della risposta:

```json
{
   "jsonrpc": "2.0",
   "id": 1,
   "result": 991991
}
```
