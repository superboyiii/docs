# Metodo getconnectioncount

Ottiene il numero attuale di connessioni per il nodo.

## Esempio

Corpo della richiesta:

```json
{
   "jsonrpc": "2.0",
   "method": "getconnectioncount",
   "params":[],
   "id": 1
}
```

Corpo della risposta:

```json
{
   "jsonrpc": "2.0",
   "id": 1,
   "result": 10
   }
```
