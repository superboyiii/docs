# `getconnectioncount`

Retorna o número atual de conexões com o nó.

Não possui parâmetros.

## Exemplo

### Requisição:

```json
{
   "jsonrpc": "2.0",
   "method": "getconnectioncount",
   "params":[],
   "id": 1
}
```

### Resposta:

```json
{
   "jsonrpc": "2.0",
   "id": 1,
   "result": 10
}
```
