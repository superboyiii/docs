# getblocksysfee メソッド

指定されたインデックスに基づいて、ブロックのシステム料金を返します。

## パラメータ説明

インデックス：ブロックインデックス

## 例

リクエスト:

```json
{
  "jsonrpc": "2.0",
  "method": "getblocksysfee",
  "params": [1005434],
  "id": 1
}
```

レスポンス:

```json
{
    "jsonrpc": "2.0",
    "id": 1,
    "result": "195500"
}
```

レスポンスの説明：

結果：ブロックのシステム料金は、NeoGasの単位で計算されます。
